# tirger
nodejs mvc frame

##start frame
1. npm install -g pm2
2. cd server
3. npm install 
3. npm run start
4. In the browser input 127.0.0.1:8083 or yourIp:8083, and then can see the page
##problem
. If start frame failed, please try "pm2 delete all", then "npm run start"
